🇺🇸 **English**

## What's new in v1.0.0
This is the first stable release of PhotoDedup! This release includes the source code to run the application locally.

### Key Features
- **Intelligent Duplicate Detection**: Find exact and visually similar duplicates using AI (MobileNetV2).
- **Google Takeout Integration**: Automatically restore lost EXIF metadata from JSON files.
- **Automatic Media Organizer**: Rename and structure your library into `/YYYY/MM/` directories.
- **Multilingual**: Interface available in English, Portuguese, and Spanish.
- **Safe Deletion**: Duplicates are moved to the Recycle Bin.

### How to use
1. Download the Source Code (`zip` or `tar.gz`) below.
2. Extract the files and open a terminal in the folder.
3. Install dependencies: `install_dependencies.bat` (Windows) or `pip install -r requirements.txt`
4. Run: `python photo_dedup.py`
